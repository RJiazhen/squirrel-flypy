# 码表优化修改记录

## 符号输入统一
以下按键统一为直接输入以下符号，而不是出现多个候选，新增一个「重置」字典，添加相关配置：
- [ 【
- ] 】
- \ 、
- / 、
- ¥ ¥
- % %
- * *

## 分号引导的快捷符号

windows 端小鹤音形默在输入分号后再搭配字母键可以输入快捷符号，需要新增「快符」字典，并添加上相关配置：

- T 制表符
- O 「」
- P 『』
- H 《》
- J “”
- L 〔〕
- N 模拟 End 键

其中 O、P、H、J 这种输入成对符号的实现可以参考 分号+K 输入括号的实现方式。

## O 符

windows 端小鹤音形默认 O 键 + E 键可引导输入以下符号，需要新增 「O符」 字典，并添加上相关配置

oea	æ ʌ ɑ:
oee	ə ɜ:
oen	ŋ
oeo	ɒ ɔ:
oer	ʒ
oes	θ
oeu	ʃ
oez	ð
oey	ɪ i:
oew	ʊ u:
